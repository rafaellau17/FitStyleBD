# FitStyleBD
FitStyle es una plataforma web diseñada para la elección de atuendos. Frente a la común problemática de la indecisión y tiempo corto al seleccionar prendas para comprar y formar un buen atuendo, el sistema da la posibilidad de al registro, completar el perfil con datos específicos que ayudarán a que brindemos recomendaciones en base a los colores que guarden armonía, ocasiones y temporadas.

La arquitectura del proyecto está basada en un modelo entidad-relación y un modelo relacional implementados en Oracle SQL, apoyado con scripts de PL/SQL que incluyen funciones, triggers y procedimientos almacenados para garantizar la integridad y reglas de negocio. La interfaz donde se visualizan las tablas está elaborada con la ayuda de ASP.NET Maker 2024, ofrece distintas pantallas donde se puede editar, eliminar y agregar elementos a las distintas tablas.
